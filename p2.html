SQL (Structured Query Language): SQL is a programming language for querying and modifying data and managing databases. .SQL was standardized first by the ANSI (American National Standard Institute) and later by the ISO (International Organization for Standardation).

MySQL: MySQL is a RDBMS (Relational Database Management System) which has more then 11 million installations. The program runs as a server providing multi-user access to a number of databases. MySQL is owned and sponsored by a single for-profit firm based in Sweden.

HTML (HyperText Markup Language): It's the authoring language used to create documents on the World Wide Web. HTML defines the structure and layout of a Web document by using a variety of tags and attributes.
